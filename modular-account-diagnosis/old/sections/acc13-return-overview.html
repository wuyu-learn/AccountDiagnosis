字段与交互逻辑（V2 重点）
  • 口径分两块（解决上下对不齐）：上块固定「累计持有收益+累计收益率」（建仓以来口径，明确标注），含今日/已实现/持有天数；下块「区间表现」随区间 tab 变（区间收益/超额/区间回撤/区间收益额）。两块各自标口径，不再混用一个收益率让客户困惑
  • 走势图专业化：面积填色折线（我的橙 #FB6340）+ 沪深300 对比线（蓝 #5B8FF9）+ 可选基准；零轴虚线；顶部标峰值点、底部标谷值点；买卖点红买绿卖 + 独立开关（可关）
  • 基准可选下拉：沪深300默认/中证500/上证50/创业板指/中证偏股基金指数/中证全债；切换重算对比线与超额
  • 去除同期基民对比：合规不允许"战胜X%同期基民/同类排名"，仅保留与市场指数基准的对比
  • 亏损安抚（合规）：属实才说——真跑赢/真长期为正/真回撤小才说对应话；长期也负不硬安慰；中性蓝、不用红绿
  • 浮动/落袋拆分（V2.1）：累计持有收益 = 浮动收益 + 已落袋（已实现），上块明示两项——回应"赚的钱到手了没"这一高频追问
  • 分年视角（V2.1 新增）：自然年切片回答"我去年赚多少"；各年相加 = 建仓以来累计（对账闭环）；首个年度标"自X月建仓起"，避免误读为全年表现
  • 可比性口径（关键正确性红线）：走势图收益率曲线采用份额归一（时间加权 TWR）口径，方能与指数公平对比；区间收益数字为客户资金口径（含申赎时点影响）。有申赎时两者会有差，该差额由「收益体验差卡（ACC-16）」承接解释，本卡不混用
合规红线：① 删除"同期基民/同类排名"对比（截图中"战胜+57.64%同期基民"类表述不合规）；② 不放广告位（截图中"止盈不止步"banner 类不进卡）；③ 安抚话术属实才说、过合规。
